To run:

 1. Clone the project at https://github.com/vergenzt/cs4641-project2.
 2. Open from Eclipse with Scala IDE installed.
 3. Run `ABAGAIL/project2.jardesc` from the workspace.
 4. `java -cp project2.jar:</path/to/scala_library_2.11>.jar opt.experiment.TestAll`

Main class is at `src/opt/experiment/Project2Experiment.scala`. Main logic for
experiment execution is at `src/opt/experiment/Experiment.scala`.

Note: Experiment progress is output to stderr, results are output to stdout.
